# Protótipo de Portfólio - Pedro Dorigatti

Este projeto é um portfólio pessoal desenvolvido como parte da disciplina **Introdução ao Desenvolvimento Web** no curso de Engenharia de Computação da **USP - São Carlos**. O objetivo principal é consolidar conhecimentos práticos de estruturação com HTML5 e estilização avançada com CSS3.

## 🚀 Sobre o Projeto
O projeto utiliza como base o template **EasyFolio** (BootstrapMade), adaptado para refletir minha trajetória acadêmica, habilidades técnicas e projetos de Iniciação Científica.

### Tecnologias e Ferramentas:
* **Linguagens:** HTML5, CSS3.
* **Framework:** Bootstrap 5.
* **Apoio:** O processo de personalização e correção de bugs contou com o auxílio de Inteligência Artificial para fins didáticos e otimização de workflow.
